# Карта проекта (Project Map)

## app/

Содержит маршруты Next.js-приложения. Главная страница загружает основной опыт SolveOS, API-маршруты обрабатывают solve, memory, outcomes, benchmarks и reviews.

## components/

Содержит видимый интерфейс. Включает консоль решений, панели результата, баннер действия, журнал, модал настроек и дашборды.

## lib/

Содержит основную логику. Включает движок решений, промпты, хранилище памяти, напоминатели действий, метки идентичности, оценку профиля, языковые вспомогательные функции и общие типы.

## scripts/

Содержит локальные тестовые скрипты. Проверяют поведение: маршрутизацию намерений, маршрутизацию обзоров, обновления профиля, петли обучения и разнообразие ответов.

## data/

Хранит локальные данные проекта. `decisions.json` содержит сохранённую историю решений, `simulations.json` — данные симуляций.

## docs/specs/

Содержит актуальные спецификации SolveOS. Файлы объясняют проект простым языком для обзора, оценки и будущей разработки.
