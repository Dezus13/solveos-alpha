# SolveOS Specs

## Where Specs Live

Active specifications live in `docs/specs/ru/`.

The root `docs/specs/` directory keeps only navigation and workflow files:

- `README.md`
- `RULES.md`
- `commit-rules.md`

## Source Of Truth

The Russian specifications in `docs/specs/ru/` are the active source of truth for this legacy spec tree.

The higher-level product specification hierarchy lives in `спецификации/` and remains authoritative for product, implementation process, technical constraints, and user stories.

Do not re-create duplicated English functional specs under `docs/specs/*.md`.

## Agent Instructions

The root agent instruction file is `AGENTS.md`.

Additional specification workflow instructions live in `спецификации/AGENTS.md`.
