# SPEC-0005: Pricing Model

## Overview
SolveOS follows a tiered subscription model based on usage volume, feature access, and agent sophistication.

## Tiers

### 1. Free
- **Cost**: €0/month
- **Includes**:
  - 3 Decisions per month.
  - Basic Decision Engine (Standard LLM).
  - Web interface access.

### 2. Pro
- **Cost**: €19/month
- **Includes**:
  - Unlimited Decisions.
  - Full AI War Room (Strategist & Skeptic).
  - Snapshot Sharing.
  - Decision History & Search.

### 3. Operator
- **Cost**: €79/month
- **Includes**:
  - Custom Agent Personas (e.g., "Analyze like Steve Jobs").
  - API Access for programmatic decisions.
  - Team Collaboration (Shared War Rooms).
  - Priority Support.

### 4. Enterprise
- **Cost**: Custom Pricing
- **Includes**:
  - SSO & Advanced Security.
  - Dedicated Agent Instances.
  - On-prem or Private Cloud Deployment.
  - Training & Onboarding.
