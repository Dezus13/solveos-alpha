# U001 — Первый запрос

## Родительская власть

Главнее этой истории: `спецификации/бизнес-контекст.md` и связанные feature specs.

Эта история — пример. Она не создает новые требования.

## Связанные спецификации

- `спецификации/бизнес-контекст.md`
- `спецификации/правила-спецификаций.md`
- `спецификации/функции/F001-чат-и-ответы/спецификация.md`
- `спецификации/функции/F003-анализ-и-рекомендации/спецификация.md`

## Границы

Внутри: пример первого запроса нового пользователя.

Вне: новые функции, новые экраны, платежи, аккаунты и техническая реализация.

## Что эта история не имеет права определять

- обязательный UI-дизайн
- формат API
- новые поля данных
- поведение, которого нет в feature specs

## Как пользователь

Я новый пользователь, который застрял в решении.

## Я хочу

Ввести ситуацию простыми словами.

## Чтобы

Быстро получить ясный ответ и следующий шаг.

## Шаги

1. Открыть главную страницу.
2. Ввести решение или проблему.
3. Отправить запрос.
4. Прочитать ответ.
5. Выполнить или запомнить следующий шаг.

## Ожидаемый результат

- пользователь понимает вердикт
- видит конкретный следующий шаг
- может задать уточнение
- решение сохраняется в историю
- страница не требует регистрации
